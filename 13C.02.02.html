Design with Frameworks

13C.02.02

Task

The assignment's task was to use a framework to create a responsive website using pregiven files.

Focus

  • Bootstrap
  • Pico.css
  • Tailwind CSS
link to github repository link to solution prototype

Process

Before actually working on this project, I played around with all three css frameworks in a sandbox to test them all and see how they work differently. I then decided to use Pico.css as a lot of styling was done automatically when the css was connected to the html.

Styling the navigation was quite easy with Pico. I used the utility class 'container-fluid' to ensure that the navigation bar was always 100vw and items were centered. In order to attain the nav bar look I was searching for I did have to add some additional div's and other elements.

Styling the UI card was not what I had expected, as Pico's layout method included using a header, body and footer element inside each card to separate the sections. I used a header to hold the image and a body element to hold the card text.

Reflection

By completing this project I gained knowledge about CSS frameworks and how to work with them to quickly and "easily" create a responsive prototype. It was fun to play around with the different frameworks and see how some were easier than others.
